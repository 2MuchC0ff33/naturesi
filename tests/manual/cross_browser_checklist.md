# Cross-Browser Manual Checklist

Browsers to test (recommended):

- Chrome (desktop & Android)
- Firefox (desktop & Android)
- Safari (macOS & iOS)
- Edge (desktop)

Viewport scenarios:

- Mobile narrow (360×800)
- Tablet (768×1024)
- Desktop (1366×768)

Checks:

- [ ] Header/nav behaviour and category links
- [ ] Cart add/remove updates badges and cart page
- [ ] Checkout summary and PayPal form render correctly
- [ ] Service worker does not break navigation when offline (test offline page)
- [ ] Ensure no console errors in each browser's devtools
